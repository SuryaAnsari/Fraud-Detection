# 💳 Credit Card Fraud Detection

This project focuses on detecting fraudulent transactions using machine learning techniques. It uses classification models along with imbalance-handling methods to identify fraud in highly skewed datasets.

---

## 📌 Project Overview

Credit card fraud detection is a challenging problem due to the extreme class imbalance (very few fraud cases compared to normal transactions).

In this project, we:
- Perform data preprocessing and exploration
- Handle class imbalance using techniques like SMOTE
- Train multiple machine learning models
- Evaluate models using appropriate metrics focused on fraud detection
